# Revival Med

Cognitive remediation tool designed for therapists.

## Description

Cognitive remediation is a process based on neuroplasticity whose objective is to reduce cognitive deficits and strengthen daily life skills.

Numerous therapies apply cognitive remediation, including RECOS (Remédiation Cognitive) and CRT (Cognitive Remediation Therapy). This tool provides a framework to support the delivery of these therapies.
